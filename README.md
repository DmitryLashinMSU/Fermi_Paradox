<h2>О проекте</h2>
<p>
    Данный проект реализует простую иллюстрацию парадокса Ферми посредством моделирования обмена сигналами 
    между цивилизациями в галактике. Приложение позволяет настроить параметры моделирования через графический 
    интерфейс, понаблюдать за появлением, исчезновением и взаимодействием очагов разума в космосе, а также 
    оценить вероятность обнаружения другой цивилизации.
</p>
